Dear Neenie & Family,

This is the Ticinese Language Encyclopedia - a special gift
to help you explore our heritage language!

WHAT YOU NEED:
- Python installed on your computer (free download)
- Windows computer
- Internet browser (Chrome, Firefox, or Edge)


INSTALLATION INSTRUCTIONS:

ONE-CLICK INSTALLATION (Recommended):

Step 1: Right-click "INSTALL_COMPLETE.bat"
  - Choose "Run as administrator"
  - This will automatically install Python if needed
  - Just wait 2-5 minutes while it works

Step 2: Desktop shortcut appears
  - The installer does everything automatically!

Step 3: Use the encyclopedia
  - Double-click the "Ticinese Encyclopedia" icon on your desktop
  - Your browser will open automatically
  - Enjoy exploring!


MANUAL INSTALLATION (If automatic doesn't work):

Step 1: Install Python manually
  - Download from: https://www.python.org/downloads/
  - Check "Add Python to PATH" during installation

Step 2: Run "INSTALL.bat"
  - Double-click "INSTALL.bat" in this folder
  - Follow the prompts

Step 3: Use the encyclopedia
  - Double-click the desktop icon


HOW TO USE THE ENCYCLOPEDIA:

1. Click on the tabs at the top:
   - Vocabulary: 1,284 Ticinese words with pronunciations
   - Stories: 10 beginner stories in Ticinese
     * Hover your mouse over any word to see its English meaning!
   - History & Culture: Learn about Ticinese heritage

2. Use the search bar to find specific words

3. When you're done, simply close the browser window


WHAT'S INSIDE:

- 1,284 vocabulary words with IPA pronunciations
- 28 pronouns with usage examples
- 18 grammar rules
- 10 A1-level comprehensible input stories
- Complete history timeline (1000 CE - present)
- 12 cultural fact categories
- Interactive hover tooltips for vocabulary


TROUBLESHOOTING:

Q: The installer says Python is not found
A: Download and install Python from python.org
   Make sure to check "Add Python to PATH"

Q: Nothing happens when I double-click the desktop icon
A: Right-click the icon and select "Run as administrator"

Q: The browser doesn't open automatically
A: Open your browser and type: http://localhost:8080

Q: I want to close the encyclopedia
A: Just close the browser window
