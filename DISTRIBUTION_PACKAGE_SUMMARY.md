## Troubleshooting Guide

### Common Issues & Solutions:

**Q: Python not found**
- Solution: Download Python from python.org
- Check "Add Python to PATH" during installation

**Q: Shortcut doesn't work**
- Solution: Right-click shortcut ‚Üí "Run as administrator"

**Q: Browser doesn't open**
- Solution: Manually open browser and go to http://localhost:8080

**Q: Port 8080 already in use**
- Solution: Close other programs using that port
- Or edit launch_encyclopedia.py to use different port
