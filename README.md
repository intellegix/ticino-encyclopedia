# Ticino Encyclopedia

**Ticinese Language Encyclopedia** - An interactive digital encyclopedia preserving the Ticinese dialect (Western Lombard) as spoken by Swiss-Italian emigrants from Canton Ticino (1850-1915).

## Features

- **1,284 Vocabulary Words** with IPA pronunciations and etymologies
- **28 Pronouns** with comprehensive usage examples
- **18 Grammar Rules** with comparative Italian analysis
- **10 Interactive Stories** (A1 level) with hover translations
- **Historical Timeline** (1000 CE - present)
- **Cultural Facts** covering traditions, cuisine, and heritage

## Technology

- Pure HTML/CSS/JavaScript (no build process required)
- JSON database files for all content
- Responsive design for mobile and desktop
- Interactive vocabulary tooltips using CSS hover effects
